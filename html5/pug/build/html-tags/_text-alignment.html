Default Paragraph (No Specified Alignment)

These alignment classes are not standard html tags but should be accounted for in a design system. This one is the default paragraph alignment. Paragraphs are usually represented in visual media as blocks of text that are separate from adjacent blocks by vertical blank space and/or first-line indentation.
